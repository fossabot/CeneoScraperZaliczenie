CeneoScraper

Cel aplikacji
Aplikacja ta ma na celu zbieranie danych produktów ze strony ceneo.pl.

Działanie aplikacji
Aby rozpocząć należy wybrać zakładkę Ekstrakcja opinii z menu górnego. W formularzu na tej stronie należy podać id produktu ze strony ceneo.pl. Po wciśnięciu przycisku rozpoczynającego ekstrakcję, nastąpi przekierowanie na stronę wydobytego produktu, gdzie można zobaczyć podstawowe dane produktu i opinie na jego temat. Listę zapisanych produktów można znaleźć w zakładce Lista produktów. Z tej listy można wybrać produkt, aby przejrzeć informacje o nim.

Zastosowane technologie
Projekt wykorzystuje języki programowania Python oraz JavaScript. Do przetrzymywania danych używana jest baza danych MySQL. Część backendowa projektu wykorzystuje framework Flask. Do łączenia z bazą danych wykorzystywany jest mysql.connector, a do wyciągania danych ze strony używana są biblioteki requests i BeautifulSoup 4. Frontend używa Bootstrapa 4, jQuery, a także biblioteki DataTables. Całość połączona jest ze sobą przy użyciu Webpacka.

Zastosowane podejścia
Stworzyłem 3 główne klasy, które pomagają mi zarządzać danymi po stronie backendu. Klasa Opinions przetrzymuje tablicę opinii złożoną z elementów klasy Opinion, a także zarządza wydobyciem danych opinii. Przy czym samo wydobycie danych pojedynczej opinii odbywa się w klasie Opinion. Klasa Product służy do przechowania danych o danym produkcie. Frontend wykorzystuje obiekty do zarządzania stanem podstron, które tego wymagają. Po przejściu na podaną podstronę inicjalizowane są te obiekty, których ona potrzebuje.
